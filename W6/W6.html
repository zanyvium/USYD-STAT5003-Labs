We may also have calculated the accuracy from the confusion matrix values. To see this, lets calculate the specificity, sensitivity, as well as \(F_1\) scores from the confusion matrix values.

SensFunc <- function(TP, FN) TP/(TP+FN)
SpecFunc <- function(TN, FP) TN/(TN+FP)
PrecFunc <- function(TP, FP) TP/(TP+FP)
RecaFunc <- SensFunc

sensitivity_10fold <- SensFunc(knn.TP, knn.FN)
specificity_10fold <- SpecFunc(knn.TN, knn.FP)
precision_10fold <- PrecFunc(knn.TP, knn.FP)
recall_10fold <- RecaFunc(knn.TP, knn.FN)
F1_10fold <- 2*precision_10fold * recall_10fold / (precision_10fold + recall_10fold)

We may create a data frame to visualise the results. Note that given that there are 208 rows in Sonar, it is no longer useful to look at the outcome on each of the \(100\) folds individually, as each only has \(\sim 2\) data points to evaluate. We thus aggregate the results.

df_ConfMatStats <- data.frame(acc = knn.accuracy_100, sens = sensitivity_100fold, spec = specificity_100fold, F1 = F1_100fold)
